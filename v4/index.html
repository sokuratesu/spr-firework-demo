Video Duration Overlay

Yuki requested that video thumbnails show the play time so users know if they're watching a 15-second clip or a 4-minute video. This demo proves it's possible using Firework's public embed API — no authentication required.

Confirmed: The Firework embed API returns a duration field (in seconds) for every video. Liam can call this same endpoint from React to fetch durations and render formatted time badges on thumbnails. No API key needed.

How it works

1
React calls the Firework public embed API with the same hashtag filter used by the embed component. The endpoint is cdn1-api.fireworktv.com/embed/v2/channel_hashtags/{channel}/feeds
2
The JSON response includes duration (in seconds as a float) plus thumbnail_url, caption, and hashtags for each video.
3
React formats and renders the duration as a badge on each thumbnail (e.g., "0:15", "1:00", "4:25"). Shorts (<60s) get a red badge.

/* Liam: React implementation pattern */

const endpoint = `https://cdn1-api.fireworktv.com/embed/v2/channel_hashtags/${channel}/feeds`;
const params = new URLSearchParams({
  page_size: 10,
  hashtag_filter_expression: hashtagExpr
});

const res = await fetch(`${endpoint}?${params}`);
const data = await res.json();

// Extract duration for each video
const videos = data.feed_items.map(item => ({
  id: item.video.encoded_id,
  duration: item.video.duration, // seconds as float
  title: item.video.caption,
  thumbnail: item.video.thumbnail_url,
  hashtags: item.video.hashtags
}));

// Format: 264.998 → "4:25"
function formatDuration(seconds) {
  const m = Math.floor(seconds / 60);
  const s = Math.floor(seconds % 60);
  return `${m}:${s.toString().padStart(2, '0')}`;
}

Floating Player Behavior

When a user opens a video in the fullscreen popup and clicks the minimize icon, the video shrinks to a floating player in the bottom corner. The user can continue browsing while the video plays. This is a built-in Firework behavior.

1
Click any video above to open the fullscreen popup.
2
Click the minimize icon in the upper corner. The player shrinks to a floating widget in the corner.
3
The user can continue browsing the page while the video plays. Click the floating player to expand it again, or X to close.
No extra code needed. The floating player is a built-in Firework SDK behavior. It works automatically with StoryBlock and other embed types. Liam does not need to implement this separately.
